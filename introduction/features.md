# Core Features

Mentor LMS provides a rich set of features designed to create a comprehensive learning environment. Below are the key features that make Mentor LMS a powerful platform for online education.

## Course Management

<!-- ![Course Management](/configuration/course-management.png) -->

- **Course Creation**: Intuitive interface for building structured courses
- **Curriculum Builder**: Organize content into sections and lessons
- **Media Integration**: Support for video, audio, documents, and interactive content
- **Course Settings**: Control pricing, visibility, and enrollment options
- **Course Preview**: Test the student experience before publishing

## User Dashboard

- **Role-based Interfaces**: Different dashboards for administrators, instructors, and students
- **Personalized Experience**: Relevant information and actions for each user type
- **Progress Tracking**: Visual indicators of course completion and achievements
- **Notifications**: Important alerts and updates in real-time
- **Account Management**: Profile settings and security controls

## Enrollment System

- **Course Catalog**: Searchable listing of available courses
- **Shopping Cart**: Multi-course purchase functionality
- **Wishlist**: Save courses for future consideration
- **Enrollment Management**: Track student registrations and access
- **Course Access Control**: Time-limited or permanent enrollment options

## Payment Integration

- **Multiple Gateways**: Support for PayPal, Stripe, Mollie, and Paystack
- **Secure Transactions**: Industry-standard security protocols
- **Currency Options**: Multiple currency support for international sales
- **Discount Coupons**: Promotional code functionality
- **Tax Handling**: Configurable tax rates and calculations

## Quiz & Assessment Tools

- **Multiple Question Types**: MCQ, true/false, matching, essay, and more
- **Quiz Timer**: Set time limits for assessments
- **Automated Grading**: Instant results for objective questions
- **Question Banks**: Create and reuse questions across courses
- **Certificate Generation**: Award certificates upon successful completion

## Discussion Forums

- **Course-specific Discussions**: Focused conversations related to course content
- **Threaded Replies**: Organized discussions with nested comments
- **Moderation Tools**: Control and review discussion content
- **Notifications**: Alert users to new posts and replies
- **Rich Text Formatting**: Support for formatted text, links, and media

## Instructor Payouts

- **Revenue Sharing**: Configurable commission rates for course sales
- **Payment Scheduling**: Regular automated payout processing
- **Earning Reports**: Detailed financial tracking for instructors
- **Multiple Payout Methods**: Various options for receiving funds
- **Tax Documentation**: Support for necessary financial reporting

## Analytics & Reporting

- **User Statistics**: Track student engagement and activities
- **Course Performance**: Measure completion rates and assessment results
- **Financial Reports**: Monitor revenue, sales, and payouts
- **Custom Reports**: Generate specific data based on your requirements
- **Data Visualization**: Graphical representation of important metrics

## Notification System

- **Email Alerts**: Automated messages for important events
- **In-platform Notifications**: Real-time updates within the system
- **Customizable Templates**: Edit notification content and design
- **Event Triggers**: Configure which actions generate notifications
- **Notification Preferences**: User control over notification frequency

## Customizable Themes

- **Visual Editor**: Front-end customization without coding
- **Brand Integration**: Add your logo, colors, and design elements
- **Responsive Design**: Optimal viewing on all devices
- **Page Sections**: Enable or disable homepage components
- **Navigation Control**: Customize menu items and structure
