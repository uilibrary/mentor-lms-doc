# Upcoming Features

This page outlines the exciting new features planned for future releases of Mentor LMS. Our development team is continuously working to enhance the platform with innovative tools and capabilities to improve the learning experience.

## Live Classes for Courses

**Coming Soon:** Integration with Zoom Meeting and Google Meet

- Seamlessly host live classes through Zoom or Google Meet directly within each course
- Schedule and manage virtual sessions from the instructor dashboard
- Students can join live classes with a single click without leaving the platform
- Access to recording capabilities for later viewing (via Zoom/Google Meet)
- Leverage existing Zoom/Google Meet features like screen sharing, whiteboard, and chat
- Calendar integration for automatic scheduling and email reminders

## Automatic Version Updates

**Coming Soon:** Streamlined system updates

- One-click updates directly from the admin dashboard
- Automatic compatibility checks before updating
- Backup creation before performing updates
- Detailed changelogs accessible within the admin interface
- Rollback capability in case of issues

## Additional Homepage Templates

**Coming Soon:** More design options for your platform's landing page

- Multiple professionally designed homepage templates
- Easy selection from the admin dashboard
- Customize each template with your content and branding
- Different layouts optimized for various types of educational institutions
- Mobile-responsive designs for all new templates

## Enhanced Customization Options

**Coming Soon:** Greater control over the look and feel of your platform

- Advanced theme customization with more color palettes
- Custom CSS injection for developers
- Expanded widget options for sidebars and content areas
- User interface element resizing and positioning

## Feedback and Requests

We value your input on which features should be prioritized in our development roadmap. If you have suggestions for additional features or improvements, please contact our support team at support@ui-lib.com with the subject line "Mentor LMS - Feature Request".

When submitting a feature request, please include:
- A clear description of the proposed feature
- How this feature would benefit you and other users
- Any examples of similar features in other platforms

Our goal is to continuously improve Mentor LMS to better serve educators and learners around the world.
