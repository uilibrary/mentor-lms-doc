# Frontend Customization

## Overview

MentorLMS includes a powerful **No-Code Drag & Drop Web Builder** that allows you to create and customize your website pages without writing any code. This visual page builder provides complete control over your site's appearance, layout, and content through an intuitive drag-and-drop interface.

## Key Features

- 🎨 **Visual Page Editor** - Edit pages in real-time with drag & drop
- 🎯 **40+ Pre-built Sections** - Ready-to-use, professional sections
- 🧩 **Component Library** - Extensive collection of draggable components
- 🎨 **Full Color System** - Customize your entire color scheme
- 📱 **Responsive Design** - Mobile-first, fully responsive layouts
- 🖼️ **Media Manager** - Upload and manage your media files
- ⚙️ **Page Settings** - Configure SEO, meta tags, and more
- 📐 **Layout Structure** - Visual page structure overview

## 1. Frontend Management Dashboard

![Frontend Dashboard](/customization/frontend/frontend-01.png)

### Features Available:

#### **Frontend Enable/Disable**

- Toggle frontend visibility on/off
- Useful for maintenance or testing periods
- Instant activation/deactivation

#### **Color System Change**

- Select the Primary color.
- The changes across entire site instantly.

#### **Page Management**

## 3. Component Library

![Component Library](/customization/frontend/frontend-03.png)

### Available Components:

#### **Layout:**

- Section, Container, 2 Columns, 3 Columns, Div, HR

#### **Elements:**

- Heading, Paragraph, Span, Image, Video, Iframe, Icon, Link, Button, Form, Input, Textarea, Select, Checkbox, Radio, Label, UI, Li, Header, Nav, Footer, Article, Aside, Dropdown, Popover, Accordion, Tabs, Drawer

### How to Use Components:

1. **Browse** the component library in the left sidebar
2. **Drag** your desired component
3. **Drop** it onto your page canvas
4. **Click** to select and edit
5. **Customize** properties in the right panel

### Section Customization:

All sections are fully customizable:

- ✏️ Edit all text content
- 🖼️ Replace images
- 🎨 Change colors
- 📐 Adjust spacing and layout
- 🔧 Modify component properties

## Workflow

### Creating a New Page:

1. Go to **Settings > Frontend**
2. Click **Create New Page**
3. Choose a template or start blank
4. Add Pre-built sections for quick layout
5. Customize content and images
6. Configure page settings
7. Preview on different devices
8. Publish when ready

### Editing Existing Pages:

1. Navigate to **Frontend Management**
2. Find your page in the list
3. Click **Edit** button
4. Make your changes in the visual editor
5. Preview changes
6. Save and publish
