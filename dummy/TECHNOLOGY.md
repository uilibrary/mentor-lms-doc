## Technologies Used

### Backend

- **Framework**: Laravel 12.x with PHP 8.2+
- **Architecture**: Modular architecture with nwidart/laravel-modules
- **Database**: MySQL/PostgreSQL with Eloquent ORM
- **Authentication**: Laravel Sanctum, Socialite for OAuth
- **File Storage**: Local/S3/Google Cloud Storage with Spatie Media Library
- **Payment Processing**: PayPal, Stripe, Mollie, Paystack SDKs
- **Email Service**: SMTP configuration with mail testing
- **Job Processing**: Laravel Queues for background tasks

### Frontend

- **Framework**: React 18.x with TypeScript
- **Build Tool**: Vite for fast development and building
- **UI Library**: Radix UI components with Tailwind CSS
- **State Management**: Inertia.js for seamless Laravel integration
- **Styling**: Tailwind CSS 4.x with custom design system
- **Rich Text Editor**: TipTap with advanced formatting options
- **Video Player**: Plyr for responsive video playback
- **Charts & Analytics**: Recharts for data visualization
- **PDF Generation**: jsPDF for certificate generation

### Additional Libraries

- **Zoom SDK**: Real-time video conferencing integration
- **CodeMirror**: Code editing and syntax highlighting
- **Date Handling**: date-fns for date manipulation
- **Icons**: Lucide React for consistent iconography
- **Form Handling**: React Hook Form with validation
- **Notifications**: Sonner for toast notifications
- **Progress Bars**: NProgress for loading indicators

### Development Tools

- **Code Quality**: ESLint, Prettier for code formatting
- **Type Checking**: TypeScript for type safety
- **Testing**: Pest PHP for backend testing
- **Package Management**: Composer for PHP, npm for JavaScript
- **Version Control**: Git with GitHub Actions for CI/CD
