# Application Update

Update your Mentor LMS application to the latest version using the built-in update system from the admin dashboard.

## Accessing the Update System

1. Log in to your Mentor LMS admin dashboard
2. Click **"Maintenance"** in the left sidebar
3. Find the **"Application Update"** card

## Important Guidelines

::: warning Critical Steps
Always follow these essential guidelines before updating:
:::

1. **Backup First**: Create a backup before updating
2. **File Format**: Upload must be a valid ZIP file from CodeCanyon
3. **System Compatibility**: Check PHP version and disk space
4. **Maintenance Mode**: Site will be temporarily unavailable
5. **Don't Interrupt**: Keep browser open during update process

## Update Process

### Step 1: Create Backup

1. Go to **"Application Backup"** section
2. Click **"Create Backup"** button
3. Wait for completion and verify in "Recent Backups"

### Step 2: Download Latest Version

1. Log in to your CodeCanyon account
2. Download the latest version ZIP file
3. Save to your local computer

### Step 3: Upload and Install

1. In **"Application Update"** section, click **"Update Application"**
2. Select the downloaded ZIP file
3. Click **"Upload"** and wait for completion
4. **Do not close browser** during the process

### Step 4: Verify Update

1. Check for success confirmation message
2. Verify new version number is displayed
3. Test key features (admin login, course creation, payments)

## What Happens During Update

- **File Extraction**: New files are installed
- **Database Migration**: Database schema is updated automatically
- **Cache Clearing**: Application caches are cleared
- **Permission Setting**: File permissions are set correctly

## Troubleshooting

### Common Issues

**Update Fails to Start:**

- Check file format (must be valid ZIP)
- Verify file isn't corrupted
- Ensure sufficient disk space

**Update Process Interrupted:**

- Don't panic - system maintains backup points
- Check for error messages
- Use restore feature if needed

**Post-Update Issues:**

- Clear browser cache
- Check file permissions
- Verify database connection

### Recovery Options

**Using Backup Restore:**

1. Go to **"Recent Backups"** section
2. Find backup created before update
3. Click **"Restore"** button
4. Wait for completion

## Best Practices

- **Schedule updates** during low-traffic periods
- **Test on staging** environment first
- **Inform users** about maintenance windows
- **Keep multiple backups** for safety

The Application Update system keeps your Mentor LMS current with the latest features and security updates while maintaining system stability.
