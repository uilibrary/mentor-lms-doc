# Installation Service

We understand that setting up a Learning Management System can be challenging, especially if you're not familiar with web application deployment. To ensure a smooth setup process, Mentor LMS offers a professional installation service by our dedicated team.

## Requesting for Installation

If you've purchased Mentor LMS and would like our team to handle the installation for you, follow these simple steps:

1. **Send an Installation Request**

   - Email our support team at support@mentorlms.com
   - Include your purchase receipt or license key in the email
   - Use the subject line: "Mentor LMS - Installation Request"

2. **Provide Temporary Access of Server**

   - Our team will respond with instructions on what access is required
   - You'll need to provide temporary credentials to your hosting server
   - This typically includes:
     - cPanel/Plesk login details or equivalent hosting panel access
     - FTP/SFTP credentials
     - Database creation privileges
   - All access information should be sent securely

3. **Installation Process**
   - Our team will install Mentor LMS on your server
   - We'll configure the basic settings to get you started
   - You'll receive a confirmation email when installation is complete

## Security Assurances

We understand that providing server access requires trust. Here's how we protect your information:

- All access credentials are handled only by senior team members
- Your credentials will be deleted after successful installation
- We recommend changing all passwords after installation is complete
- No sensitive information is stored in our systems

## After Installation

Once installation is complete:

1. You'll receive an email with:

   - Admin login URL and temporary credentials
   - Basic setup instructions
   - Recommendations for next steps

2. We recommend:
   - Changing the admin password immediately
   - Updating all temporary access credentials
   - Following our post-installation checklist

## Eligibility

This service is available to all customers who have purchased a valid license for Mentor LMS. There is no additional charge for standard installation on compatible hosting environments.
